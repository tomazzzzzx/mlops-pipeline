# Mlops Pipeline

End-to-end MLOps pipeline: data validation → training → deployment.

## Stages
1. **Data Validation**: Schema checks, drift detection
2. **Training**: Distributed with experiment tracking
3. **Evaluation**: Automated quality gates
4. **Deployment**: Canary rollout, A/B testing
5. **Monitoring**: Latency, throughput, data drift

## Integrations
MLflow, Weights & Biases, S3, Kubernetes, Prometheus
